משתמשים והרשאות
📞 ניהול מענה טלפוני (ימות המשיח)
שליטה בממסרים דרך שלוחת ימות המשיח — לכל מתקשר מספר ID פרטי (לא נחשף ב-Git) ולכל ID הרשאות ציבוריות נפרדות.
מתקשרים מורשים
בודק שכל המתקשרים המוגדרים כאן מורשים בכלל להיכנס לקו בימות (שכבה נפרדת מההרשאות שלנו) — ומוסיף אוטומטית את מי שחסר.
🧪 בדיקת endpoint /yemot
שולח קריאה אמיתית ל-/yemot — בדיוק כמו שיחת טלפון לכל דבר. הממסר שנבחר באמת יופעל/יכבה (וייתועד ביומן וב-Timeline), אז אפשר לבדוק הרשאות והודעות בלי לחייג בפועל.
🔊 עריכת הודעות קוליות (TTS) ו-ext.ini בשלוחה
עריכה ישירה של קבצי הטקסט/TTS בשלוחת ימות המשיח דרך ה-API הרשמי (DownloadFile/UploadFile) — בלי להיכנס לאתר ימות. אפשר גם לעדכן אוטומטית את כל קבצי תפריט "ממסר ← פעולה ← משך זמן" לפי שמות הממסרים והרשאות המתקשרים שמוגדרים כרגע במערכת.
📋 יומן שיחות IVR
כל ניסיון שיחה — גם נכשל/לא מורשה — יתועד כאן. מוצגים רק ID ושם, ללא מספר הטלפון עצמו. ייעשה שימוש ב-addServerLog הקיים עם תיוג מקור "ימות".
— יוצג לאחר חיבור לשרת —
🏠 התקני Home Assistant
הוסף התקנים מ-Home Assistant לרשימת הממסרים — כך תוכל לתזמן אותם בדיוק כמו בקרי Tasmota. כל התקן שתוסיף יקבל מזהה ממסר ייחודי ויהיה זמין בבחירת ממסר בתוכניות.
⚙️ הגדרות חיבור
הטוקן נשמר מקומית בשרת ואינו נחשף ב-Git. ב-HA: Settings → שם פרופיל (מטה) → Long-Lived Access Tokens → Create Token
התקנים זמינים ב-HA
סמן התקנים להוספה לרשימת הממסרים. התקנים שכבר נוספו מסומנים.
